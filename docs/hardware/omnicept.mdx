# HP Reverb G2 Omnicept

## Summary

The HP Reverb G2 Omnicept Edition is a VR headset that features built-in eye tracking. It is compatible with VRCFaceTracking via the [VRCFTOmniceptModule](https://github.com/200Tigersbloxed/VRCFTOmniceptModule).

## Set-up

Setup of the VRCFTOmniceptModule is fairly simple. Please read carefully to avoid any issues. 

:::note
This guide assumes you have HP Glia Runtime/SDK installed.
If you do not have the [HP Glia Runtime/SDK](https://developers.hp.com/omnicept/downloads) installed, please install it before continuing.
:::

## Using The Omnicept Module

To use Omnicept eye tracking with VRChat, you will need to use the [VRCFTOmniceptModule](https://github.com/200Tigersbloxed/VRCFTOmniceptModule/releases) with the VRCFaceTracking software. To download this, go to VRCFaceTracking's Module Registry and install the VRCFTOmniceptModule. Once installed, you will need to restart VRCFaceTracking.

The VRCFTOmniceptModule is compatible with the following hardware:

| Headset | Eye Tracking | Facial Tracking |
| ---------------------------------- | -------- | --------- |
| HP Reverb G2 Omnicept Edition                         | `✅` Yes | `🛈` Via the Project Babble Module  |

Assuming there are no connection issues with the hardware, the connection hardware components should be initialized by VRCFaceTracking after a few seconds.

## Uninstalling The VRCFTOmniceptModule

To uninstall The VRCFTOmniceptModule, go to the Module Registry, select the VRCFTOmniceptModule and at the bottom click the uninstall button.
